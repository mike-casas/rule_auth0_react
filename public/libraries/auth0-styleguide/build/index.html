    Fork from github Download Sketch

    Getting Started

    There are few ways to initialize depending on the use case.

    From CDN

    Using our CDN is the simplest solution. Just add the following line in your project's HTML head and you are done.

    <link href="https://cdn.auth0.com/styleguide/4.4.24/index.css" rel="stylesheet" />

    Installing from Github

    Clone the repository to your computer. If you want more information read here.

    git clone https://github.com/auth0/styleguide.git
cd styleguide 
grunt dev
